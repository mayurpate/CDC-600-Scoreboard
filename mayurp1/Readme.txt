-----------------------------------------ACA Spring 2017 Project--------------------------------------------------------------

Please extract the provided zip folder in your working directory. 

This project is developed in Python 2.7 and only contains single file. This program only need python 2.7 to be installed. The submitted file is executable file. You may need to provide execution previlleges on your system. 

For providing execution permission, please use following command:

$ chmod +x scoreboard.py

For running this program, use following command from the directory where project is extracted:
	
	./scoreboard.py inst.txt data.txt config.txt result.txt

This project will generate output in result.txt file in required format.
